# 💍 Site d’Invitation de Mariage Islamique – Diaporama Immersif

Un site web moderne, élégant et immersif pour annoncer un mariage islamique, avec un **diaporama animé plein écran** et des **animations fluides** basées sur Anime.js.

---

## 🎯 Objectif

Ce projet vise à créer une **expérience visuelle émotionnelle et immersive**, inspirée des sites premium de mariage, pour présenter :

* Les noms des mariés
* Les dates de cérémonie
* Le lieu
* Une ambiance romantique et islamique

## ✨ Fonctionnalités

* 🎞️ **Diaporama immersif plein écran**

  * Images animées (effet Ken Burns : zoom + déplacement)
  * Transitions fluides (fade + blur)
  * Lecture automatique en boucle

* 🎬 **Animations avancées avec Anime.js**

  * Apparition lettre par lettre des textes
  * Animations synchronisées avec les images
  * Timeline fluide et professionnelle

* 💡 **Effets visuels**

  * Overlay sombre élégant
  * Animations au scroll
  * Hover sur boutons

* 📱 **Responsive Design**

  * Adapté mobile, tablette et desktop

## 🚀 Installation et utilisation

1. **Cloner ou télécharger le projet**

```bash
git clone https://github.com/ton-projet/mariage-site.git
```

2. **Ajouter les images**

* Placer les photos du couple dans le dossier `assets/`
* Respecter les noms :

  ```
  couple1.jpeg → couple5.jpeg
  ```

3. **Ouvrir le projet**

* Double-cliquer sur `index.html`
  OU
* Utiliser un serveur local :

```bash
python -m http.server
```

Puis ouvrir :

```
http://localhost:8000
```

## 🎞️ Fonctionnement du diaporama

Le diaporama utilise une **timeline Anime.js** pour :
